Installation
============

To install this module, you need to:

1.  Clone the branch 12.0 of the repository https://github.com/OCA/hr
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Apps -> Apps*
5.  Search For *HR Employee Emergency Contact*
6.  Install the module

Usage
=====

To use this module, you need to:

1. Open employee data
2. Go to *Emergency Contact* tab
3. Add/select emergency contact
