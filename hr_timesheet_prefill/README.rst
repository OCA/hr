Add a wizard to create in mass timesheets lines for employee/project for a given date range

**Table of contents**

.. contents::
   :local:

Usage
=====

On the main timesheet menu, you have now a new entry "timesheet prefill".
When this wizard is open, you can fill for who/what you want to create some timesheets in one shot.
